# Resume Website

This is a simple single-page resume website created using plain HTML and CSS.  
The layout is clean and minimal, focusing on proper structure, readability, and basic web standards.

The page includes sections for personal details, skills, experience, projects, education, and contact information.

## Technologies Used
- HTML5
- CSS3

## Setup and Usage
- Clone or download the repository
- Open the `index.html` file in a browser to view it locally
- Edit the content in `index.html` to update details

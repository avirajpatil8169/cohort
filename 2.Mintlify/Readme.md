# Mintlify Landing Page Clone

A recreation of a modern documentation platform landing page built with HTML and CSS.

## Sections Recreated

- **Header Navigation** - Logo, nav links, and action buttons
- **Hero Section** - Main headline with email capture form and background image
- **Documentation Preview** - Interactive mockup showing sidebar navigation and content area
- **Trust Logos** - Company logos section
- **Feature Sections** - Two alternating layout features with images
- **Enterprise Grid** - Three-column feature cards
- **CTA Section** - Call-to-action with background image
- **Footer** - Multi-column footer with links and social icons

## Design System

### Fonts
- **Primary Font**: Inter (300, 400, 500, 600, 700 weights)
- **Icon Library**: Font Awesome 6.4.0

### Colors
- Primary Dark: `#0f2e37`
- Primary Green: `#24c39e`
- Primary Green Hover: `#1ea384`
- Text Dark: `#0f172a`
- Text Gray: `#64748b`
- Background Light: `#f8fafc`
- Border Color: `#e2e8f0`
- White: `#ffffff`

## Technical Details

- Pure HTML/CSS (no JavaScript)
- Flexbox and CSS Grid layouts
- CSS custom properties for theming
- Background images from Unsplash
- Responsive container (max-width: 1280px)

## What I Learned

This project helped me understand modern layout techniques, design systems with CSS variables, and how to create a profesional landing page structure. Working with flexbox and grid together, implementing hover effects, and creating visual heirarchy through typography and spacing were key learnings.
